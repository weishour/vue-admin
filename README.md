## 描述

[vue-admin](https://www.weishour.com/) 唯守vue前端框架.

## 安装依赖

```bash
$ npx pnpm add -g pnpm
$ pnpm i
```

## 运行

```bash
$ pnpm run serve
```

## 打包

```bash
$ pnpm run build
```
